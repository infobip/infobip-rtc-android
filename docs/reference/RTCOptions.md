## `static builder()`

### Description
Creates builder instance used to build new instance of `RTCOptions`.

### Arguments
none

### Returns
[`RTCOptions.Builder`](./RTCOptions.md) - Instance of builder.

### Example
```
RTCOptions.Builder builder = RTCOptions.builder();
RTCOptions rtcOptions = builder.build();
```
